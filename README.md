# 🌞 Solaris-CET

**Solaris-CET** este un asistent AI pentru energie verde, panouri solare și tehnologii sustenabile.

## ✨ Caracteristici

- 🤖 **AI Integrat** - Rulează direct în browser folosind Transformers.js și TinyLlama
- 💯 **100% Gratuit** - Fără chei API, fără costuri lunare
- 🔒 **Privat** - Datele tale nu pleacă niciodată de pe dispozitiv
- ⚡ **Rapid** - Model optimizat pentru browser (60MB)
- 🌐 **Disponibil Online** - Găzduit gratuit pe GitHub Pages
## 💎 Tokenomics & TON Integration
Solaris-CET nu este doar un asistent, ci nucleul economic al comunității Cetățuia.
* **Token:** CET (Solaris Cetățuia)
* **Blockchain:** TON (The Open Network)
* **Supply:** 9,000 unități (Hyper-scarce)
* **Utilitate:** Deținerea token-ului oferă acces la modulele de AI de înaltă precizie pentru agricultura regenerativă și prioritizarea procesării în unitățile locale din Puiești.

## 🏗️ Arhitectura Sistemului (Local-First AI)
Datele tale nu părăsesc niciodată browserul. Fluxul de date este următorul:
1. **User Input** (React UI) → 2. **Web Worker** (Thread separat) → 3. **Transformers.js** (Inferență) → 4. **Local Model** (TinyLlama în Browser Cache).
- React 18 + TypeScript
- Transformers.js (Hugging Face)
- TinyLlama 1.1B Chat Model
- GitHub Pages (Hosting)
